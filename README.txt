# particle_reco
Particle mass reconstruction using Cern's ROOT and CMSSW

===================================================================

Cut_optimization: Contains programs for optimizing cuts to maximize signal significance. More documentation within the directory.

LambdaCData: Contains CMSSW analyzer code for CRAB tasks.

rooFitHisto_error.cpp: A macro for fitting peaks in TH1D objects. See the documentation within the file for instructions on use.
